# onxticker
an amazing little library to standardize onixcoin pricing

##  install
npm i -g onxticker

## use lib
https://runkit.com/hbastidas/runkit-npm-onxticker

## terminal use
$ onxticker vef
or
$ notify-send "$(onxticker usd)"
